# Code of Conduct

## Our pledge

We want this project to be a harassment-free space for everyone, regardless of experience level, background, or identity. We pledge to act with respect toward other contributors and maintainers.

## Expected behavior

- Be respectful in issues, pull requests, and discussions.
- Give constructive feedback - critique the code or the idea, not the person.
- Accept that maintainer time is limited; a slow response isn't a rejection.

## Unacceptable behavior

- Harassment, insults, or personal attacks, public or private.
- Trolling, deliberately derailing discussions, or spamming issues/PRs.
- Publishing others' private information without consent.

## Enforcement

If you experience or witness unacceptable behavior in this repository, report it by opening a private report via [GitHub's "Report content" feature](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) on the offending issue/comment/PR, or by contacting the repository owner ([@V1rg1lee](https://github.com/V1rg1lee)) directly through their GitHub profile.

Maintainers may edit, remove, or lock content, and may ban contributors who violate this Code of Conduct, at their discretion.

## Scope

This Code of Conduct applies within all project spaces (issues, pull requests, discussions) and when representing the project elsewhere.

This document is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
